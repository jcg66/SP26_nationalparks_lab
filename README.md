# Android Lab 3 - *SP26_nationalparks_lab (Parks Explorer)*

Submitted by: **James Girgis**

**Parks Explorer** is a simple Android app that allows users to scroll through a list of beautiful parks and learn more about each one.

Time spent: **5** hours spent in total

## Required Features

The following **required** functionality is completed:

- [x] **Live data is loaded from the National Parks API**
- [x] **Parks are displayed using a RecyclerView**
- [x] **Park images are downloaded and displayed using Glide**

The following **additional** features are implemented:

- [x] Each park displays additional information such as the weather at the location
- [x] User can move forward/backward through "pages" of park items
